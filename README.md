# PDF Chapter Splitter

根据 PDF 书签（目录）信息，智能识别 **书籍 → 卷 → 章** 的层级结构，将 PDF 按章拆分为多个小文件。

## 功能特性

- **智能层级识别** — 通过正则匹配书签标题，自动分类为书籍、卷、章
- **层级回退** — 关键词无法匹配时，根据书签缩进层级自动推断
- **精准分页** — 章的起始页 = 书签所在页，结束页 = 下一书签起始页前一页
- **跳过无关条目** — 附录等非书/卷/章条目不拆分，但参与边界计算；章的子标签完全忽略
- **自动清空输出** — 每次拆分前自动清空输出目录，避免残留旧文件
- **文件名含页码** — 输出文件名包含页码范围和页数，方便查阅

## 关键词匹配规则

| 层级 | 匹配模式 | 示例 |
|------|---------|------|
| 书籍 | `第X册`、`第X部` | 第一册、第叁部 |
| 卷 | `第X卷`、`第X篇` | 第二卷、第壹篇 |
| 章 | `第X章`、`第X回`、`第X节`、`第X讲`、`第X课`、`第X话` | 第三章、第1回、第十二节 |
| 跳过 | 含"附录"的条目 | 附录 朱由检大事记 |
| 忽略 | 章的子标签（层级更深） | 章下面的小节 |

> 其中 X 支持中文数字（一二三...）、大写数字（壹贰叁...）和阿拉伯数字。
> 不匹配任何模式的条目默认跳过（不拆分但参与边界计算）。

## 环境要求

- Python ≥ 3.13
- [uv](https://github.com/astral-sh/uv) 包管理器

## 安装

```bash
git clone <repo-url>
cd PDF-Chapter-Splitter
uv sync
```

## 使用方法

```bash
# 基本用法（输出到 PDF 同目录下的 <文件名>_split/ 文件夹）
uv run main.py <PDF文件路径>

# 指定输出目录
uv run main.py <PDF文件路径> <输出目录>
```

### 示例

```bash
uv run main.py 三国演义.pdf
uv run main.py "/path/to/明朝那些事儿.pdf" ./output
```

输出：

```

输出目录: 三国演义_split
```

## 输出文件命名

```
序号-书籍名-卷名-章节名-页码信息.pdf
```

| 场景 | 示例文件名 |
|------|-----------|
| 书籍 + 卷 + 章 | `001-第一册-第一卷-第一章 绑定-p.1-15(15页).pdf` |
| 卷 + 章 | `001-第一卷-第一章 绑定-p.1-15(15页).pdf` |
| 仅章 | `001-第一章 绑定-p.1-15(15页).pdf` |

书籍名、卷名为空时自动省略。
